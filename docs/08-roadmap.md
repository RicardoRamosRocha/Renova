# Roadmap de Desenvolvimento

## Fase 0 — Fundação

Objetivo: Estruturar o projeto.

Entregas:

* Solução .NET
* Clean Architecture
* PostgreSQL
* ASP.NET Identity
* Controle de acesso
* Documentação
* GitHub

Prazo estimado: 1 a 2 semanas

---

## Fase 1 — CRM

Objetivo: Gestão de alunos e familiares.

Entregas:

* Cadastro de alunos
* Cadastro de familiares
* Histórico do aluno
* Agendamentos
* Atendimentos
* Dashboard inicial

Prazo estimado: 4 a 6 semanas

---

## Fase 2 — Prontuário

Objetivo: Controle clínico.

Entregas:

* Anamnese
