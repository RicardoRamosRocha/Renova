# LGPD e Segurança

## Objetivo

Garantir proteção dos dados pessoais e sensíveis tratados pela plataforma Renova.

## Dados Tratados

* Nome
* CPF
* Data de nascimento
* Telefone
* Endereço
* Dados clínicos
* Evoluções
* Prontuários
* Histórico de atendimento

## Princípios

* Finalidade
* Necessidade
* Transparência
* Segurança
* Responsabilização

## Controle de Acesso

Perfis distintos para:

* Administrador
* Coordenador
* Profissional
* Atendente
* Professor
* Aluno
* Familiar

## Segurança

* ASP.NET Identity
* Senhas criptografadas
* HTTPS obrigatório
* Auditoria de ações
* 2FA para usuários internos

## Auditoria

Serão registrados:

* Login
* Logout
* Inclusões
* Alterações
* Exclusões
* Acessos a prontuários

## Backup

* Backup diário
* Backup semanal completo
* Recuperação em caso de falha
