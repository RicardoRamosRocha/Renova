# Renova

Sistema CRM + EAD para Recuperação e Apoio à Dependência Química.

## Sobre o Projeto

O Renova é uma plataforma desenvolvida para clínicas, comunidades terapêuticas, profissionais e instituições que atuam no acompanhamento e recuperação de pessoas com dependência química.

A solução integra CRM, Prontuário Eletrônico, Plataforma EAD, Gestão Familiar, Atendimento, Financeiro e Relatórios em um único ambiente seguro e escalável.

## Objetivos

* Centralizar informações dos alunos e familiares.
* Organizar atendimentos e acompanhamentos.
* Disponibilizar conteúdo educacional online.
* Automatizar processos financeiros.
* Garantir conformidade com a LGPD.
* Preparar a expansão para aplicativo mobile.

## Tecnologias

* ASP.NET Core 9
* Blazor Server
* Entity Framework Core
* PostgreSQL
* ASP.NET Core Identity

## Arquitetura

* Clean Architecture
* API REST
* Banco de Dados PostgreSQL
